== display 속성 ==
1) block요소 (블록요소)
- 자동 줄바꿈, 세로정렬
- 기본 가로너비값을 전체 100%로 인식(부모상속)
- width, height 공간값 적용 가능.
- <div> <p> <ul> <li> <ol> <dl> <dt> <dd> <h1 ~ h6> 

2) inline요소 (인라인요소)
- 가로정렬, 줄바꿈안됨, 글자처럼 인식함
- 기본 가로너비값을 자신의 안쪽에 작성된 글자 or 이미지양 만큼 인식
- width, height 공간값 적용 안됨.
- 주로 내용(컨텐츠) 담당할때 사용하는 속성.
- <a> <span> <strong> <i> <em> ...

3) {display:inline-block}
- 기본은 inline요소
- 인라인요소 이면서 block의 (가로, 세로 공간값 인식) 특징가짐.
- float대신 대체해서 사용하지는 않음.
  (가로배열 할떄 특정영역에서 사용)
- <img> <input> ...

4) {display:none}
- 해당 속성을 가진 태그는 html구조상에서 삭제되는 형식.
- 공간도 없어지고 ui 화면출력도 되지않는 상태.
- 주로 ui user interaction 효과에 사용함.

5) {display:block} 은 2가지 형식으로 사용됨.
 (1) 인라인 요소인 태그를 block요소화 시켜 공간값을 인식시키는 개념.
 (2) {display:none} 으로 사라진 태그를 {display:block} 으로 보이게 할때 사용.
