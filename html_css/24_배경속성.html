d1
1) background-color 속성 : 배경색상만 줌
d2
2) background-image 속성 : 배경 이미지만 ㄴ{background-image:url(경로);}
=> 기본적으로 가로와 세로의 공간이 있어야 정확히 배경 이미지가 출력됨
=> 기본적으로 가로 세로 무한반복해서 이미지 출력.
d3
3) background-repeat 속성 : 배경이미지 반복여부 설정
ㄴ값
no-repeat, repeat-x, repeat-y
d4
4)background-position : 배경이미지 출력 위치
- 기본 (0,0) 왼쪽 상단.
- 배경 이미지가 반복하고 있다면 쓸수없음.
- {background-repeat:no-repeat}일때 함께 사용가능.
d5
5)background 속성 : 1~4 개별속성을 한번에 처리할때 쓸수 있는 속성.
ㄴ{background: 색상 url(이미지) 반복여부(repeat) 가로위치 세로위치;}
*값을 순서대로 작성하는것이 중요함.
*4가지 속성값을 모두 작성하지 않아도 됨.(작성한 부분만 적용)
ㄴ안되는 경우는 모두 풀어서 적어야함.
d6
6) background-size : 배경이미지 크기 조절
{background-size: 가로크기 세로크기}
=> {background-size: 150px auto;} => 가로크기150px 세로는 이미지 비율에 맞춰서 인식함
d7
7) background-attachment 속성 : 배경이미지의 스크롤 방법을 지정하는 속성.
{background-attachment: fixed} : 해당 태그의 영역에서만 배경이미지가 출력.
웹브라우저 기준 배경이미지를 고정시킴.






css 이미지 스프라이트 기법 (image sprite)

  1. 여러개의 이미지를 하나의 이미지로 합쳐서 사용하는 방법 => 이미지의 관리 용이 / 웹사이트 이미지 로딩시간 및 용량 줄여줌 => 이미지 적용시 웹표준을 준수하여야 함으로 가상대체 텍스트를 사용해야함. (IR: Image Replacement 기법 사용)
